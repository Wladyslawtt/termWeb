# Terminal Portfolio Website

A modern, interactive terminal-style portfolio website that showcases your bio and links in a unique way.

## Features

- Terminal-like interface with command history
- Interactive commands to display different sections of your portfolio
- Responsive design that works on all devices
- Custom styling with a modern look
- Command history navigation using arrow keys

## Available Commands

- `help` - Display available commands
- `about` - Learn about me
- `skills` - View technical skills
- `projects` - See projects
- `contact` - Get contact information
- `clear` - Clear the terminal

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Start typing commands to explore the portfolio

## Customization

To customize the content:

1. Open `script.js`
2. Modify the content in the `executeCommand` function
3. Update the personal information in each case statement

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
